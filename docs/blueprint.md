# **App Name**: DevConnect

## Core Features:

- Snippet Generation Tool: Use AI to provide users with the ability to create code snippets from natural language descriptions.
- User Authentication: Allow users to register and log in via Google or GitHub.
- Snippet Display: Display code snippets in a readable format with syntax highlighting, supporting multiple languages.
- User Profiles: Display user information and links to snippets.
- Follow/Unfollow: Allow users to follow each other
- Snippet Bookmarking: Enable users to bookmark interesting and/or useful snippets for future reference.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm and professional look.
- Background color: Light gray (#F5F5F5) for a clean, code-friendly interface.
- Accent color: Orange (#FFB74D) to highlight important actions.
- Body font: 'PT Sans' sans-serif for a clear and readable interface.
- Headline font: 'Space Grotesk' sans-serif for headers and titles.
- Use a consistent style of icons with labels to clearly indicate each action and category.
- Provide feedback with smooth and unintrusive animation.
